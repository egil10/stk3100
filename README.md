## About

This repository contains solutions and materials for the mandatory assignments in STK3100. The course covers regression analysis techniques beyond standard linear regression, including models for binary/binomial, Poisson, and gamma distributed response variables, as well as methods for dependent responses.

## Repository Structure

- **oblig1/** - First mandatory assignment
  - R scripts for problem solutions
  - Data files (titanic.txt, wingspan.txt)
  
- **oblig2/** - Second mandatory assignment
  - R scripts for problem solutions
  - Data files (fertility_data.csv)
  - Generated plots (residuals vs fitted plots)

- **rsc/** - Course resources and reference materials

## Course Information

- **Credits:** 10
- **Level:** Bachelor
- **Teaching:** Autumn
- **Language:** English

The course requires 2 mandatory assignments to be approved before sitting the final exam.
